User manual - Admin side

To navigate to the admin side of the product, simply follow the link at the bottom of any page on the client side.

Menu

The menu at the top of the admin page should easily describe any of the tasks that you might want to do. Clicking on one of these menu items loads content relevant to that task.

Add product

Clicking this will load content that will allow you to add a new product.

Validation is performed on each of the inputs here, and feedback is given if you do something wrong.

To upload a product image, click the ‘choose file’ button, and once you have selected the file that you want to upload, press the ‘upload’ button. If you submit a product without uploading a product image, then a default image will be used as a placeholder until you change it on the Update Product page.

Picking a category for a product will mean that it will be shown in that category when selected from the menu on the client side.

Update product

Clicking this will load content that will allow you to update a product.

Validation is performed on each of the inputs here, and feedback is given if you do something wrong.

Selecting the product that you wish to update will load all of the current values of that product. Make the changes that you require, and then press submit to make the changes.

To upload a product image, click the ‘choose file’ button, and once you have selected the file that you want to upload, press the ‘upload’ button.

If you submit a product without uploading a product image, then the image that was previously used for that product will remain.

Remove product

Clicking this will load content that will allow you to remove a product from the database.

Simply select the product that you want to delete, and press submit to remove it.

Adding a category

Clicking this will load content that will allow you to add a category.

Type the name that you want to call the category, and then choose a parent for that category. Selecting root will create a new main node for other categories, and will be displayed on the client side as an addition to the menu bar.

Choosing something other than Root will make your new category a child, and will be displayed on the client side when you hover over the category that you chose.

Removing a category

Clicking this will load content that will allow you to delete a category.

Selecting the category that you want to delete and pressing submit will delete that category. You have to be careful here, as deleting a parent category will mean that all of the children for this category will not be able to be shown on the client side, making navigation to products harder.

User manual - Client side

To navigate to the client side from the admin pages, simply follow the link saying ‘shop home’ at the bottom of the page.

Menu

The menu at the top of the page forms as the main navigation for the customer. Hovering over a main category reveals the sub categories, that can be clicked to reveal products in that category.

Recently added

This area will display the 30 most recently added products

Search

This search bar will enable you to search through every product that exists in the database, and will show all of these products in a drop down list. Clicking on an item in the list will take you to the product page for that item.

Product page

Clicking on any of the products will take you to a page for that item, which will give you a description for that product, the stock, and the price. If the stock value is 0, then a message will appear saying that there is no stock for that item.
