Specification

Admin

  • Each input in the admin pages has validation in it to prevent certain SQL injection techniques which would break the client side, or could delete entire databases
  • The validation also makes sure that only relevant data can be added, for example, you can only add numbers for the price
  • You are able to update product data, so if you make a mistake, or have to alter stock levels, you can quickly and easily do that
  • Easy to use navigation
  • Intuitive design

Client

  • Has a limit on the number of products that can be displayed on a page. This means that if you had a million products, it wouldn’t try to load them all
  • A search function to make navigation easier
  • Search functionality prevents SQL injection, so a customer can’t delete databases or alter product prices
  • Automatically generated menu, which makes maintenance easy
  • Easy navigation
  • Intuitive design
  • Clear and consistent page layout
